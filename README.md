# VR Tire Manufacturing Industry Simulation 🚀

[cite_start]Proyek simulasi Virtual Reality (VR) yang memvisualisasikan alur produksi ban secara interaktif[cite: 84]. [cite_start]Dikembangkan sebagai media edukasi imersif untuk memahami tahapan manufaktur di lingkungan pabrik yang aman[cite: 53, 54].

## 🛠️ Teknologi & Tools
* [cite_start]**Game Engine**: Unity 3D [cite: 87, 120]
* [cite_start]**3D Modeling**: Blender (untuk mesin dan lingkungan pabrik) [cite: 91, 120]
* [cite_start]**VR Framework**: XR Interaction Toolkit & Oculus SDK [cite: 90, 120]
* [cite_start]**Bahasa**: C# Scripts [cite: 89, 120]
* [cite_start]**Hardware**: Meta Quest / Oculus Quest [cite: 88]

## 📋 Alur Simulasi
* [cite_start]**Pojok K3**: Prosedur wajib menggunakan helm safety sesuai standar industri[cite: 94, 105].
* [cite_start]**Gudang Bahan**: Observasi bahan baku seperti karet mentah dan aditif kimia[cite: 95, 106].
* [cite_start]**Ruang Produksi**: Interaksi memasukkan bahan ke mesin menggunakan snapzone dan aktivasi conveyor[cite: 96, 97, 110, 111].
* [cite_start]**Loading Area**: Pemindahan ban jadi ke rak penyimpanan dan box pengiriman[cite: 99, 100, 115].
